## Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/KianHmz/tic-tac-toe-react.git

```


2. **Install dependencies**
```bash
cd tic-tac-toe-react
npm install

```


3. **Run the app**
```bash
npm run dev

```


*The app will open at `http://localhost:5173`.*
